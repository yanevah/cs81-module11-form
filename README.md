# Module 11 - Contact Form

## Overview
This project shows how a React contact form is built using controlled components and useState. The application validates user input and dynamically displays submitted data.

## Features
- Controlled form inputs
- Real-time state management with useState
- Basic form validation
- Conditional rendering of submitted data

## Technologies
- React
- JavaScript (ES6+)
- HTML5
- CSS
